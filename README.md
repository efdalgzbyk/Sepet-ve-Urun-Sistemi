# Sepet-ve-Urun-Sistemi
Bu proje, Java’da OOP mantığını öğrenmek amacıyla hazırlanmış basit bir sepet uygulamasıdır.
## Kullanılan Konular

Class ve Object

Encapsulation

Inheritance

Abstract Class

Interface

ArrayList

## Proje Yapısı

Urun.java → Ürün sınıfı

SepetUrun.java → Sepete eklenen ürün

Indirim.java → İndirim hesaplama

Main.java → Programın çalıştığı sınıf

## Çalışma Mantığı

Ürünler sepete eklenir

Toplam fiyat hesaplanır

Şartlara göre indirim uygulanır
